web페이지 만들기

사용기술:Java/JSP/HTML5/CSS3/Javascript/SQL-OracleDB

웹페이지 화면 및 기능

공통

1.메인페이지
	1.1.로그인 전
		-로그인 페이지 이동 버튼
		-상품목록 페이지 이동 버튼
	1.2.로그인 후
		-상품목록 페이지 이동 버튼
회원 

2.로그인 페이지
	-id/pw 입력 칸 만들기
	-로그인 처리 기능
	-회원가입 페이지 이동 버튼
	-로그인 실패 시 알림
		-id/pw입력오류
		-미입력 로그인
3.회원가입 페이지
	-사용자 입력 칸
	-입력된 정보 DB에 입력
	-완료 페이지 이동
	-취소 (메인페이지/로그인페이지로 이동)
4.로그인/회원가입/회원정보 수정 완료 페이지
	- 로그인 완료 시 (주문내역,회원정보 수정 버튼)
	- 회원가입 완료 시(메인페이지 이동,로그인 페이지 이동 버튼)
	- 회원정보 수정 시(마이페이지 이동)
5.회원정보 수정 페이지
	-pw/주소/연락처 수정 입력칸
	-수정 버튼(DB에서 입력된 정보로 수정작업 처리)
	-취소 버튼(마이페이지로 이동)
	-탈퇴 버튼(입력된 정보 DB에서 삭제, 세션에서 삭제)
6.주문내역 페이지
	- 회원일 경우 회원정보와 일치하는 주문내역 불러오기
	- 비회원일 경우 연락처/주문비밀번호를 입력 받음
	- 해당 정보로 주문 내역 불러오기

상품

7. 상품 목록 페이지
	-상품 등록/편집 페이지 이동 버튼
	-DB에 저장된 상품들 불러오기/출력
	-상품 상세정보 페이지 이동 버튼
		-해당 상품의 정보 입력 받아 상품페이지로 전송
	-장바구니 담기/장바구니 DB에 상품정보 추가
8. 상품 등록 페이지
	- 사용자 입력으로 상품 정보 입력
	-입력 받은 정보 DB에 입력 처리
	-상품목록 페이지 이동 버튼
9. 상품 편집 페이지
	-상품 등록/목록 페이지 이동 버튼
	-DB에 등록된 상품 불러오기/출력
	-각 상품에 수정 삭제 버튼
	-수정 버튼 
		클릭시 해당 상품의 정보를 입력 받아 상품 수정페이지에 전송/ 상품 수정페이지로 이동
	-삭제 버튼
		클릭시 해당 상품의 정보를 입력 받아 해당 정보 DB에서 삭제 처리
10. 상품 페이지
	-  상품목록 페이지의 상세보기 버튼으로 상품 정보 입력 받은 정보를 불러오기/ 출력
	-  상품목록 페이지 이동 버튼
